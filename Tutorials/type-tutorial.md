# Add type in () for year 1 emoticons
8 or 0 or symbol and - or symbol(  add type here....)

Use Ctrl+C to copy to clipboard.
# Example:
<(Hello World)
# Hint
Rename 8 or 0 or symbol to symbols in ().
# Features:
* String method -()
* For 8 or 0 or symbols only (not for text)
* Text requirements:
  * V: :(Hello World!)
  * X: Hello World(:)
* () is good for ()
* And a () () for () ()!
